APIs for ExchangeLah application

This environment is launched with Elastic Beanstalk .NET Core on Linux Platform. Check out AWS Elastic Beanstalk overview and AWS Elastic Beanstalk concepts for more info.

Working APIs:

GET ALL

  1. /api/module
  2. /api/basket
  3. /api/country
  4. /api/degree
  5. /api/region
  6. /api/university
  7. /api/review
  8. /api/user
  9. /api/degreeuser
    • This table identifies the primary and secondary degree for a given user.

  5. /api/region/{regionid}
  6. /api/university/{universityname}
  7. /api/review/{reviewid}
  8. /api/user/{userid}
